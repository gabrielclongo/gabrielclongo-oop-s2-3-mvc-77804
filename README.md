How to Use the Application

After running the application, follow these steps to explore all features:

 1. Login

Use one of the seeded accounts:

Admin: admin@test.com
 / Admin123!
Faculty: faculty@test.com
 / Faculty123!
Student: student@test.com
 / Student123!
 
 2. Admin Workflow

Login as Admin to manage the system:

Go to Courses
Create new courses
Go to Students
Add student profiles
Go to Enrollments
Link students to courses (IMPORTANT)
Go to Assignments
Create assignments for courses
Go to Exams
Create exams and optionally release results
Go to Assignment Results / Exam Results
Add grades for students

 3. Faculty Workflow

Login as Faculty:

View students enrolled in courses
Manage:
Assignments
Exam results
Access Gradebook to see student performance

 4. Student Workflow

Login as Student:

View personal academic data only
Access:
Gradebook
Assignment results
Exam results (only when released)

 Important Notes
 
Students must be enrolled in a course to appear in the gradebook
Without enrollments, no grades will be visible
Exam results are hidden until ResultsReleased = true
